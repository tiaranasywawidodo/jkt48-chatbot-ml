# Chatbot Informasi Umum JKT48 Berbasis Machine Learning

## Deskripsi Project
Project ini merupakan Ujian Akhir Semester (UAS) mata kuliah Machine Learning.
Sistem yang dibangun adalah chatbot informasi umum JKT48 yang mampu memahami
pertanyaan pengguna berdasarkan intent menggunakan pendekatan Machine Learning
(supervised learning), bukan rule-based murni.

Chatbot ini dapat menjawab berbagai pertanyaan seputar:
- Informasi umum JKT48
- Profil dan aktivitas JKT48
- Informasi member dan generasi
- Jadwal event dan pertunjukan
- Informasi teater JKT48
- Harga dan cara pembelian tiket
- Jam operasional teater
- Promo dan event khusus
